1-justify-content: It defines the alignment along the main axis. It helps distribute extra free space leftover when either all the flex items on a line are inflexible, or are flexible but have reached their maximum size. 2-align-items: The align-items property specifies the default alignment for items inside a flexbox or grid container. In a flexbox container, the flexbox items are aligned on the cross axis, which is vertical by default (opposite of flex-direction). 3-Gap: The gap property defines the size of the gap between the rows and between the columns in flexbox, grid or multi-column layout. 4-flex-Direction: The flex-direction property specifies the direction of the flexible items. 5-flex-wrap: flex-wrap property is used to specify whether flex items are forced into a single line or wrapped onto multiple lines. The flex-wrap property allows enabling the control direction in which lines are stacked.
